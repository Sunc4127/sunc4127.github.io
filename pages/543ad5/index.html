# 第一个方法论：TBEI (Targeted Behavioral Event Interview / 目标-行为-事件 访谈法)

在面试中面试官会这么提问：

（1）就某个真实经历进行挖掘性提问；

（2）很多为什么，询问背后的动机和逻辑；

（3）追问有没有更好的方法，这就是全部？等问题；

（4）求证你的特质，这是你真实想法吗？你确定是这样吗？

问题举例：

  1. 你在过去的项目过程中，你认为最有成就感的事情是什么？最有挑战的事情 是什么？你担任 用 STAR 方法，准备 3 个左右小故事。

  2. 怎么做的？为什么这么做？有更好的方法吗？可以改进的地方？ 准备：用到的框架、技术，进行复盘。

  3. 追加问题：这就是全部吗？还有吗？为什么？你确定吗？（典型的压力问题）

# 采用 STAR 方法

  • Situation（情景）事件的所发生的背景状况。
  • Task（任务） 任务和角色。
  • Action（行动）如何操作与执行任务。（三段论）
  • Result（结果）该项任务在行动后所达到的效果（尽量量化）。

用 STAR 方法整理过去做过 2-3 的项目，还要进行复盘（review）

# 关键面试问题

为什么想来我们公司？ 为什么想来这个岗位？（你的目的，你凭什么来）

  • 我觉得这个岗位是、、、，这个方向能够比较好的发挥我的能力和经验，实现我的职业目标

  • 我认为我的学习背景和实践能力，具备在、、、、职位发展的能力

为什么去留学

  • 因为墨尔本大学在计算机方向具有一定的先进性

为什么想回国

  • 在上学的时候就已经决定要回国发展
  • 我认为中国的高科技和互联网领域未 来对我更有利
